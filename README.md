# 🍽️ 时令健康食谱

一个每日更新时令健康食谱的网页应用，为你提供营养均衡的一日三餐搭配。

## ✨ 功能特性

### 核心功能
- **🍂 时令水果推荐** - 根据当前月份推荐当季水果
- **🍳 一日三餐搭配** - 早餐、午餐、晚餐营养均衡
- **📊 营养总览** - 显示每日总热量和营养成分
- **🔄 每日更新** - 每天自动更新食谱
- **🎨 灵动交互** - 流畅的动画和简洁的设计

### 食谱特点
- ✅ 符合时令季节
- ✅ 营养均衡搭配
- ✅ 低卡健康选择
- ✅ 简单易做
- ✅ 食材易获取

## 📊 营养标准

每日食谱按照以下标准设计：
- 总热量：1000-1200 大卡（可根据需求调整）
- 蛋白质：60-80g
- 碳水化合物：120-150g
- 脂肪：30-40g

## 🛠️ 技术栈

- **React 19** - UI 框架
- **Tailwind CSS** - 样式框架
- **esbuild** - 构建工具
- **GitHub Pages** - 部署平台

## 📝 使用说明

1. **打开网页** - 访问部署地址
2. **查看今日食谱** - 自动显示当日三餐
3. **点击换一换** - 更换今日食谱搭配
4. **参考做法** - 按照步骤制作美食

## 🌟 健康建议

- 💧 每天喝水 1500-2000ml
- 🚶 每天运动 30 分钟
- 😴 每晚睡眠 7-8 小时
